## 手机号的注册以及验证码的注册


<input type="text" id="phone">

如何知道输入的是手机号呢？

用   正则表达式

手机号规则：
	开始    1      3,4,5,7,8    9个数字   结束
	 ^	1      [3|4|5|7|8]   \d{9}	$


var reg = /^1[3|4|5|7|8]\d{9}$/

var str = $("phone").val();

$("phone").change( function() {
	if ( reg.test(str) ) {
		验证成功
	} else {
		您输入了错误的手机号码
	}

})
