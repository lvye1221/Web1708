## 纪律
### 代码格式 （10分）
1. 文件名不能出现中文
2. 样式必须放到 css 文件夹
3. js 必须放到 js 文件夹内
4. 上线目录必须按 "校区-姓名" 例如： "cs-xujia" 【区分大小写，全部用小写】
5. 文件名都要用小写字母开始，采用 驼峰命名法

### 每天工作内容（10分）
1. 每天至少一次传到 github 中  （组员-> 组长-> github中）
2. 每天早上写计划，晚上写总结
